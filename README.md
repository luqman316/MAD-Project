# QuickCart - Flutter Ecommerce App with Admin Panel

## Project Overview
QuickCart is a mobile application designed to provide users with a seamless online shopping experience. It includes a mobile app for customers and a web-based admin panel for store owners.

## Technologies Used
- **Frontend:** Flutter (Mobile and Web)
- **Backend:** Firebase (or Node.js)
- **Database:** Firestore (or any preferred database)
- **State Management:** Provider or Riverpod
- **Other Tools:** Git, GitHub, VS Code

## Features
### Mobile App:
- Browse products by category.
- Search for specific products.
- Add products to the cart and place orders.
- Track order status and view order history.

### Admin Panel:
- Manage products and inventory.
- Process customer orders.
- Generate sales reports and analytics.
